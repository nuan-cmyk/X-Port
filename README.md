# X-Port: Xray-core Proxy Manager

**X-Port** — это кроссплатформенный менеджер для управления процессами Xray-core. Он позволяет легко поднимать локальные прокси-порты (SOCKS5/HTTP), подключаться к внешним узлам VLESS, автоматически парсить подписки и визуализировать сетевой трафик в реальном времени.

Проект состоит из мощного бэкенда на **FastAPI** (Python) и красивого, отзывчивого фронтенда на чистом **HTML/Vanilla JS** с использованием Tailwind CSS.

---

## 🚀 Возможности
- **Мульти-порты:** Запуск множества независимых локальных портов с разными настройками и узлами одновременно.
- **Умный роутинг:** Поддержка правил маршрутизации (Global, Rule-based, Direct).
- **Подписки:** Автоматический импорт и фоновое обновление узлов из стандартных ссылок-подписок (VLESS).
- **График реального времени:** Отображение скорости скачивания и отдачи прямо из Xray API.
- **Фоновая служба:** Удобный скрипт для автоматической установки программы как скрытой фоновой службы (Systemd для Linux, Task Scheduler/NSSM для Windows).

---

## 🛠 Требования
1. Установленный **Python 3.10+**.
2. Скачанный исполняемый файл **Xray-core** (xray.exe для Windows или xray для Linux). Скачать можно с [официального репозитория Xray-core](https://github.com/XTLS/Xray-core/releases).

## ⚙️ Установка и Запуск

### 1. Подготовка
Склонируйте репозиторий и перейдите в папку `backend`:
```bash
git clone https://github.com/yourusername/VpnSelfhost.git
cd VpnSelfhost/backend
```

### 2. Автоматическая установка (Рекомендуется)
В проекте есть скрипт `install.py`, который автоматически создаст виртуальное окружение, установит все зависимости и добавит сервер в автозагрузку как фоновую службу.

Для установки выполните:
```bash
python install.py
```
После завершения установки сервер будет запущен в фоне. Вы можете открыть панель управления по адресу: **http://127.0.0.1:8000**

*Чтобы удалить службу из системы, выполните:*
```bash
python uninstall.py
```

### 3. Ручной запуск (для разработки)
Если вы не хотите устанавливать фоновую службу и хотите видеть логи в консоли:

1. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   # Для Windows:
   .venv\Scripts\activate
   # Для Linux:
   source .venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите сервер:
   ```bash
   uvicorn main:app --reload --host 0.0.0.0 --port 8000
   ```

## 🖥 Использование
1. Откройте **http://127.0.0.1:8000** в браузере.
2. Перейдите во вкладку **Settings** и укажите полный путь к вашему файлу `xray` (например, `C:\xray\xray.exe`). Нажмите "Save Configuration".
3. Перейдите во вкладку **Nodes** и вставьте вашу ссылку на подписку VLESS, нажмите Import.
4. Перейдите во вкладку **Ports Manager** и создайте новый порт (например, 1080), выбрав узел и протокол (SOCKS/HTTP).
5. Нажмите переключатель на карточке порта, чтобы запустить прокси. Теперь вы можете настроить ваш браузер или систему на использование `127.0.0.1:1080`.
